Resumen del proyecto:


El proyecto consiste en un sistema que permita almacenar y gestionar la información de una empresa encargada de asesorar y capacitar en prevención de riesgos.
Se pide, por lo tanto, gestionar la información relacionada a la gestión de recursos humanos. Se ha solicitado la creación de un listado de segmentos en versión HTML, 
basados en 3 grandes puntos principales:

1- Administración de asistentes a capacitaciones.
2- Login.
3- Listados de accidentes.
Para ello se implemento un diseño web el cual consta de diversas páginas, construidas solo en su forma visual (diseño del sistema en base a páginas HTML), y no de las 
funcionalidades asociadas al sistema, a excepción de las funciones de validación o acciones embebidas en una librería.
Durante el desarrollo del sistema, se utilizaron conceptos y fundamentos pertenecientes a:
-	HTML.
-	CSS. 
-	Java Script.
-	Responsive.
-	GitHub.
Todos los códigos presentes fueron codificados mediante el entorno de desarrollo Visual Studio Code y almacenados en un repositorio en Github.

Aspectos relevantes durante el desarrollo del sistema:

De acuerdo con solicitado para dar solución al problema, es correcto señalar que las mayores relevancias y complejidades presentes 
durante el proceso estuvieron asociadas a la codificación en Java Script. Esto debido a la necesidad de incorporar funciones de validación para realizar el Login en el sistema,
 donde debió considerarse de forma 
simultánea tanto el Rut del usuario como su contraseña, permitiendo la evaluación y función de ambos parámetros en forma conjunta al momento de realizar el ingreso.
